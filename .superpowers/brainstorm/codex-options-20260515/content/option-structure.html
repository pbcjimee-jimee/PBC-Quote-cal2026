A. Separate Option Groups

Each option has its own materials, labour inputs, formula results, min/max selection, and option total. The main final total never includes option totals.

B. Optional Rows In Same List

Add a flag to each material/labour row. Easier UI, but weaker when Option 1 and Option 2 need different scopes or separate formula choices.

C. Manual Option Totals

Fastest to build: type option name and amount manually. It does not reuse the calculator, so it is less useful for pricing paint/labour accurately.

My recommendation

Use A. It keeps options separate from the accepted quote price, supports Option 1 / Option 2 / Option 3 naturally, and lets each option be calculated with the same pricing rules as the main quote.
